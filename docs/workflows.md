# Workflows

## Prepare
Search source folders, build a queue preview, and create jobs.

## Packing
Build release output from prepared jobs.

## Posting
Use the configured provider builder in Settings to manage one or more posting providers, then upload packed jobs from Posting. For providers after Provider 1, **Prioritize jobs up to (GB)** can be used to prefer smaller jobs on specific providers first. Providers set to **0** stay in the same default availability pool as Provider 1.

## Share
Submit posted or imported NZBs to configured destinations. Active Share jobs can be cancelled or removed from the Share page, and the result is kept in Share History.

## Clean
Preview candidates before deleting or recycling files.
