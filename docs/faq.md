# FAQ

## Is the Freeimage API key a TMDB key?

No. It is only for **freeimage.host**.

## What happens if I leave the end tag unchanged?

PrepaC uses **PrepaC** as the default release tag.

## Can Share import files with different names?

Yes. PrepaC can pair mass imports by filename and by template content.

## Do I need to re-enter Provider 1 and Provider 2 credentials after upgrading?

No. Existing Provider 1 and Provider 2 setups carry forward automatically into the provider builder.


## What does “Prioritize jobs up to (GB)” mean?

For providers after Provider 1, this value can be used to prefer smaller jobs on specific providers first. Lower non-zero values are tried first for jobs that fit within that size. A value of **0** keeps the provider in the same default availability pool as Provider 1.
